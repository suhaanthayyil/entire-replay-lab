# Entire Replay Lab v0.1.23

Registry-accurate all-agent eval support.

## What Changed

- Made `entire eval run --agent all` match Entire's current user-facing
  built-in coder registry.
- Removed the internal test-only coder from normal all-agent eval expansion,
  docs, doctor output, and the binary all-agent fixture.
- Added a regression test that compares Replay Lab's all-agent list against
  Entire's registered coder list.
- Refreshed the patch artifact and pinned SHA-256.

## Why It Matters

`--agent all` should mean every normal coder supported by the current Entire CLI
build. This release prevents Replay Lab from accidentally advertising or
evaluating internal-only test agents, and it creates a guardrail for future
Entire agent additions.
