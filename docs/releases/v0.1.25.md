# Entire Replay Lab v0.1.25

Automated build-lock concurrency verification.

## What Changed

- Added `scripts/check-build-lock.sh`.
- The new fixture launches two `build-cli.sh` runs concurrently, waits until the
  first one holds the build lock, then verifies both complete successfully.
- The fixture also checks that the lock is cleaned up and the resulting binary
  still exposes `entire replay` and `entire eval`.
- Wired the fixture into CI, release checks, smoke checks, command docs,
  acceptance docs, testing docs, and repo verification.

## Why It Matters

v0.1.24 added the build lock after a parallel local verification run exposed a
real race in the shared patched CLI checkout. This release turns that proof into
a permanent gate so the race cannot quietly return.
