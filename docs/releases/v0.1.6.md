# Entire Replay Lab v0.1.6

Command-surface verification for the Replay Lab prototype.

## What Changed

- Added `scripts/check-command-surface.sh`.
- Added `make command-surface`.
- Replaced duplicated command checks in `smoke.sh` and `release-check.sh` with
  the shared script.
- Added an explicit CI gate for Replay/Eval command help surfaces.
- Documented the command-surface check in acceptance, command, testing, and
  README docs.

## Why It Matters

Replay Lab depends on the patched `entire` binary exposing `replay` and `eval`
commands. This release makes that user-facing command surface a first-class
verification gate instead of an inline script detail.
