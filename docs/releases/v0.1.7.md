# Entire Replay Lab v0.1.7

Stricter command-surface verification for public Replay Lab flags.

## What Changed

- Expanded `scripts/check-command-surface.sh` to verify required command flags.
- `entire replay checkpoint --help` must expose `--agent`, `--model`,
  `--test-cmd`, `--keep-worktree`, `--json`, and `--timeout`.
- `entire eval run --help` must expose checkpoint selection, agent/model,
  test, worktree, JSON, and timeout flags.
- `entire replay report` and `entire eval report` must expose `--json`.
- Updated README, command, acceptance, and testing docs to describe the stricter
  gate.

## Why It Matters

The earlier command-surface gate proved that Replay/Eval commands existed. This
release also proves the important public flags are still present, so a future
patch drift cannot silently remove the workflows documented for users.
