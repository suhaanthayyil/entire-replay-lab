# Entire Replay Lab v0.1.5

Patch-surface verification for the Replay Lab prototype.

## What Changed

- Added `scripts/verify-patch-manifest.sh`.
- Added `make patch-manifest`.
- Wired patch manifest validation into repo verification, release checks, and CI.
- Documented the patch manifest check in acceptance, command, testing, release,
  and README docs.

## Why It Matters

Replay Lab is carried as a patch against the Entire CLI. This release makes the
patch surface machine-checked, so accidental unrelated upstream edits are caught
before build or review.
