# Entire Replay Lab v0.1.22

CI gate for all-agent eval verification.

## What Changed

- Added `scripts/check-all-agent-eval.sh` to GitHub Actions CI.
- Updated repo verification so it fails if the CI workflow drops the all-agent
  eval fixture step.
- Updated the pull request template to ask for the fixture when eval or agent
  behavior changes.

## Why It Matters

v0.1.21 added the binary-level all-agent eval fixture and wired it into local
release and smoke checks. This release closes the remaining automation gap by
making GitHub Actions run the same gate on every push and pull request.
