# Entire Replay Lab v0.1.3

Release bookkeeping hardening for the Replay Lab prototype.

## What Changed

- Added `scripts/validate-release-docs.py`.
- Wired release-doc validation into repo verification, release checks, CI, and
  the Makefile.
- Removed hardcoded release-note version checks from `verify-repo.sh`.
- Updated command and testing docs to describe release-doc validation.

## Why It Matters

Replay Lab now has multiple prototype releases. This release prevents changelog
sections and `docs/releases/v*.md` files from drifting apart as the repo keeps
improving.
