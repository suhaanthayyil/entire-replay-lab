# Entire Replay Lab v0.1.12

All-release GitHub body verification for Replay Lab releases.

## What Changed

- `scripts/verify-release-state.sh` now compares every published GitHub release
  body with its matching `docs/releases/<version>.md` file.
- Mismatches still print a unified diff for the specific release that drifted.
- Release docs now describe body verification across all published releases.

## Why It Matters

The previous verifier caught drift in the latest GitHub release body. This
release extends that protection to the full release history, so older published
notes cannot silently diverge from the checked-in release record.
