# Entire Replay Lab v0.1.4

Post-release state verification for the Replay Lab prototype.

## What Changed

- Added `scripts/verify-release-state.sh`.
- Added `make release-state`.
- Documented the post-release check in the command, testing, release, and README
  docs.
- Wired the new script into repository file verification.

## Why It Matters

Replay Lab now publishes tagged prototype releases. This release adds a direct
check that release notes, local git tags, and published GitHub releases match
after publishing, so the public repo cannot quietly drift out of sync.
