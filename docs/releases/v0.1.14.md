# Entire Replay Lab v0.1.14

Project metadata and MIT license hygiene validation.

## What Changed

- Added `scripts/validate-project-metadata.py`.
- The validator checks README badges, prototype positioning, MIT license text,
  contribution expectations, and security/data-handling references.
- Wired metadata validation into `verify-repo`, release checks, CI, Makefile,
  and command/testing/release docs.

## Why It Matters

Replay Lab is meant to be easy for Entire to inspect, reuse, or adapt. The repo
already used MIT, but this release turns that from a manual convention into a
checked invariant so the project does not drift into unclear or conflicting
licensing language.
