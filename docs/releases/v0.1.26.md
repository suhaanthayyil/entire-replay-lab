# Entire Replay Lab v0.1.26

Build-lock edge-case verification.

## What Changed

- Extended `scripts/check-build-lock.sh` to verify stale lock cleanup.
- Added active-lock timeout verification so a held lock fails clearly instead of
  hanging forever.
- Kept the existing concurrent-build proof and binary command-surface check.
- Updated command docs and repo verification to require the stronger lock
  fixture.

## Why It Matters

The build lock protects the shared patched CLI checkout. v0.1.25 proved the
happy path. This release proves the failure paths too: abandoned lock
directories are cleaned up, and genuinely held locks time out with a clear
error.
