# Entire Replay Lab v0.1.28

Command-reference launchable-agent guard.

## What Changed

- Fixed the `entire replay checkpoint` docs so `--agent` lists the full
  launchable runner set: `claude-code`, `codex`, `copilot-cli`, `cursor`,
  `factoryai-droid`, `gemini`, and `opencode`.
- Added a script-hygiene validation guard that fails if the command reference
  drifts back to an incomplete launchable agent list.

## Why It Matters

v0.1.27 expanded Replay Lab beyond the original three launchers. The generated
CLI help was correct, but the manual command reference still showed the old
three-agent shorthand. This keeps the docs aligned with the actual MVP surface.

## Known Limitations

- This remains a prototype patch repo, not an upstream Entire CLI release.
- Live replay still requires each selected coding agent to be installed and
  authenticated locally.
- Pi is included in all-agent eval output, but skipped until a safe
  non-interactive launch contract is available.
