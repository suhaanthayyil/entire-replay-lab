# Entire Replay Lab v0.1.1

Docs and release hardening for the Replay Lab prototype.

## What Changed

- Added `scripts/validate-doc-links.py`.
- Wired local Markdown link and anchor validation into `verify-repo.sh`.
- Added Markdown link validation to `release-check.sh`.
- Added a visible GitHub Actions step for Markdown link validation.
- Added a `make doc-links` shortcut.
- Updated the README script index and changelog.

## Why It Matters

Replay Lab is meant to be easy to inspect, clone, build, and demo. This release
catches broken local docs links before they reach GitHub, which keeps the repo
cleaner for reviewers and future contributors.
